# Diabetes-risk-detection-using-machine-learning

Diabetes-risk-detection-using-machine-learning
Diabetes Risk Prediction Web App built using Flask and Machine Learning (Random Forest). Users can input clinical parameters and get instant risk assessment.

🩺 DiabeatAI – Diabetes Risk Prediction System
DiabeatAI is a Machine Learning-based web application that predicts the likelihood of diabetes using user-provided health parameters. The system is built using a trained ML model and deployed with a Flask backend and interactive frontend.

🚀 Features
🔍 Predicts diabetes risk (High / Low)
⚡ Instant results using trained ML model
🧠 Uses clinical parameters for prediction
🎯 User-friendly UI with smooth navigation
🔐 No data storage – privacy-focused
🧪 Input Parameters
The model uses the following medical features:

Pregnancies
Glucose Level
Blood Pressure
Skin Thickness
Insulin Level
BMI (Body Mass Index)
Diabetes Pedigree Function (family history)
Age

HTML

Scikit-learn

📂 Project Structure
🧠 How It Works User enters medical details in the form Data is sent to Flask backend via POST request Backend processes input using trained ML model Model predicts diabetes risk Result is displayed on UI

📊 Model Information Algorithm Used: Random Forest / Logistic Regression Dataset: PIMA Indians Diabetes Dataset Output: Binary Classification (High Risk / Low Risk)
